## Goal

Add a Rust workload implementing the runtime contract on host port **4203**, matching the example identity payload in `specs.md` Step 01.

## Why this step exists

Rust is suggested for Runtime, Secrets, Storage, and Memory. A contract-compliant Rust demo validates systems-language readiness and matches the spec’s example `demo-rust-api` response.
