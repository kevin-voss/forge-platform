## Goal

Fail the full suite if any platform service is not exercised by at least one demo product, so
coverage cannot silently regress.

## Dependencies

* `56.01`; [`service-coverage-matrix.md`](../../../demo-projects/service-coverage-matrix.md).

## Scope

* Machine-readable service list (from the matrix) checked against the union of each run product's
  `demo.json.services`.
* On a full run, any uncovered service → gate failure with a clear message naming the service.
* Emit a coverage table into the run report.
