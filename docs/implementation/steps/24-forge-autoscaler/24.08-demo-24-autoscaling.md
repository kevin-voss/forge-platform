## Goal

Prove Forge can autoscale workloads, workers, and infrastructure nodes in local mode with provider-neutral resources and safety controls.

## Scope

* HTTP application load test.
* Queue-backed worker backlog test.
* Node scale-up from unschedulable demand.
* Node scale-down after workload drain.
* Manual override and metric-outage fallback checks.

## Demo flow

```text
Apply ScalingPolicy for invoice-api
-> generate HTTP load
-> replicas increase
-> stop load
-> replicas decrease after stabilization
-> publish queue backlog
-> workers increase and drain backlog
-> deploy workload exceeding cluster capacity
-> Node Autoscaler asks Infrastructure for another Docker runtime node
-> workload becomes Running
-> demand decreases
-> idle node drains and is deleted
```
