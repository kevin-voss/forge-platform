## Demo flow

```text
Start three runtime nodes
-> each receives node certificate and overlay address
-> WireGuard peers converge
-> deploy frontend and api on separate nodes
-> frontend calls api.production.demo.svc.forge
-> apply NetworkPolicy allowing frontend -> api
-> verify request succeeds
-> remove allow rule
-> verify connection denied and logged
-> stop one node
-> peer and endpoint are removed
```
