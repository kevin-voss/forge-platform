## Architecture

Local development uses Docker Compose. Host tooling uses Make. Infrastructure uses documented ports (see `docs/operations/ports.md`).

```text
make / scripts
      │
      ▼
Docker Compose
├── PostgreSQL     :5001
├── NATS           :5002 / :5003
├── OCI registry   :5000
├── OTEL Collector :4317 / :4318 / :13133
├── Prometheus     :3001
├── Tempo          :3002
├── Loki           :3003
└── Grafana        :3000
```
