## Demo scenario

```text
1. create bucket "artifacts"
2. upload 50 MB streamed object
3. download it (streamed) and byte-compare
4. verify SHA-256 via ETag matches client hash
5. GET Range bytes=0-1023 → 206, exactly 1024 bytes
6. issue GET token ttl=1s, sleep, use → 401 token_expired
7. delete object → 204; GET → 404
8. docker compose restart → re-upload persists / bucket survives
```
