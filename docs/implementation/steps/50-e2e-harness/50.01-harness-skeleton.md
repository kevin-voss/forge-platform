## Goal

Stand up `tests/e2e/` as a Playwright + TypeScript project that runs headed by default and headless
on demand, records trace/video/screenshots, and has a home for per-product specs and artifacts.

## Dependencies

* None beyond a running platform (`make dev`). No other harness step precedes this.

## Scope

* `tests/e2e/` npm project with pinned `@playwright/test` and TypeScript.
* `playwright.config.ts`: `headless` from `HEADLESS`/`CI` env (default headed, `slowMo` when headed);
  `trace: 'on'`, `video: 'on'`, `screenshot: 'only-on-failure'`; `outputDir: artifacts/`.
* Directory scaffold: `harness/`, `projects/`, `artifacts/` (gitignored).
* `make e2e-install` (browsers) placeholder target documented (not full orchestrator yet).

* Any product spec, findings collector, orchestrator, or platform preflight (later steps).
