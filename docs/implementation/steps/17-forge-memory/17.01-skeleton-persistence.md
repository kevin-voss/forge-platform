## Goal

Stand up the `forge-memory` Rust (Axum) service with health endpoints, structured logging, graceful shutdown, a Docker image, Compose wiring on host port `4303`, and a durable persistence root — with no collection/vector semantics yet.

## Why this step exists

Vector storage needs a running, contract-compliant service and a persistent on-disk root before collections and search are added. Establishing the persistence layout first makes durability testable early.
